# Reference: Airbnb — Card Grids & Search

**Source**: airbnb.com (2 screenshots: experiences search, full listing page)
**Relevance**: MEDIUM — card grids, search/filter UI, clean layouts

## Layout Pattern
- **Centered top nav**: logo left, category icons centered (Homes, Experiences, Services), user actions right
- **Full-width search bar**: rounded pill shape, segmented (Where, Date, Who), pink search button
- **Content**: full width, scrollable card rows

## Search Bar
- Rounded pill: border-radius 40px, white bg, subtle shadow
- 3 segments divided by thin vertical lines
- "Where" / "Date" / "Who" labels (12px bold)
- Placeholder text below each label
- Circular pink search button on right

## Category Navigation
- Icons (48px) with labels below
- Active: underlined, slightly larger
- "NEW" red badge on some items

## Card Grid (Experiences)
- **Horizontal scroll row** with left/right arrows
- Card: image (16:10 ratio, rounded 12px), title, location, price, rating
- "Original" badge overlaid on image (dark pill)
- Heart/favorite icon on top-right of image
- Time badge on event cards ("11 AM", "2 PM")

## Calendar Dropdown
- Clean calendar grid with rounded day cells
- Today highlighted in dark circle
- Selected range highlighted in pink/red
- Quick select: "Today", "Tomorrow", "This weekend"
- Month/year header with arrows

## Key Patterns to Adopt
- **Pill-shaped search/filter bar**
- **Horizontal scrolling card rows** with navigation arrows
- **Image cards** with overlay badges and favorite buttons
- **Clean calendar UI** with quick-select options
- **Category icons** in horizontal row for filtering
