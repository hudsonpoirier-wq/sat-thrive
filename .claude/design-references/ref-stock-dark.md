# Reference: Stock Dashboard (Dark) — NVIDIA

**Source**: Stock analysis app (1 screenshot: NVIDIA detail page)
**Relevance**: LOW-MEDIUM — dark data dashboard pattern

## Layout
- **Full dark background** (#0a0a0a or #111111)
- **Top toolbar**: back arrow, stock name + logo, action icons
- **Chart area**: price chart with time range selectors (1D, 1W, 1M, 3M, YTD, 1Y, 5Y, All)
- **Metric row below chart**: horizontal scrollable KPI cards
- **Tab navigation**: News, KPIs, Earnings, About

## Price Display
- Current price: huge white text ($131)
- Change: colored badge (+$4.65 / +3.67% in green, -$1.99 / -1.52% in red)
- After hours price beside

## Analyze Dropdown
- Multi-column checkbox grid for overlays
- Categories: "Key data points", "Analyst Estimates"
- Items: Share Price, Revenue Actuals, Volume, Market Cap, EPS, P/E Ratio, etc.
- Purple checkboxes
- Action buttons: Cancel, edit, Compare, Analyze

## KPI Row
- Horizontal cards: Mkt cap, P/E ratio, FY Revenue, EPS, Gross Margin, etc.
- Small label on top, bold number below
- Minimal border, dark card backgrounds

## Key Patterns to Adopt
- **Dark dashboard** with high-contrast data
- **KPI metric row** — horizontal scrollable stat cards
- **Chart with time range selectors** as pills/tabs
- **Multi-select overlay/filter** dropdown
