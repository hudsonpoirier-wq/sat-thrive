# Reference: Hers & Hims — Clean Minimal

**Source**: forhers.com / forhims.com (2 screenshots: Hers landing, Hims landing)
**Relevance**: HIGH — clean whitespace, category cards, soft design language

## Layout Pattern
- **Full-width**, no sidebar
- **Minimal top nav**: logo left, "LOG IN" / "My account" right, hamburger menu
- **Ticker bar** at very top: scrolling feature highlights (free shipping, pricing, etc.)
- Very generous whitespace — sections breathe

## Hero Section
- **Rotating headline** with serif/light font: "Anxiety relief" / "Weight loss" (faded, animated)
- **Bold subheading**: "personalized to you" in dark sans-serif
- Subtitle: "Customized care starts here" in muted gray
- No CTA button in hero — categories below serve as entry points

## Category Cards Grid
- **2 rows**: top row 4 cards, bottom row 3 cards + 1 "Browse all" card
- Card style:
  - Rounded 16-20px corners
  - Soft beige/cream background (#f5f0e8 / rgba(245,240,232))
  - Title on top-left with colored keyword: "Lose *weight*" (green), "Reduce *anxiety*" (teal)
  - Arrow (>) on top-right
  - Product image centered, floating effect
  - "Popular" badge (green pill) on featured card
- Bottom row cards are shorter/compact
- "Browse all treatments" card: text + search icon, no image

## Color Palette
- Background: white (#ffffff) or very light warm gray
- Card backgrounds: warm beige/cream
- Text: dark charcoal (#1a1a1a)
- Accent keywords: muted green, teal, amber, coral — each category has its own accent
- Very muted, earthy, premium feel

## "Care that evolves" Section
- Full-width muted blue/gray background
- Centered serif headline
- Category pill badge above heading
- Person image below

## Key Patterns to Adopt
- **Feature/category grid cards** with soft backgrounds and accent colors
- **Minimal, premium feel** — lots of whitespace
- **Accent per category** — different color for each subject/feature
- **Soft rounded corners** on everything
- **No heavy borders** — use background color for card separation
- **Product/feature images** floating inside cards
