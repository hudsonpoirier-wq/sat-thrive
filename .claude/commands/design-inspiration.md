# Design Inspiration

When creating or modifying any UI in The Agora Project, reference the design system and visual inspirations documented in `.claude/design-references/`. These references define the look and feel we are targeting.

## How to use this skill

1. Read `.claude/design-references/design-system.md` for the extracted design principles
2. Review individual reference files in `.claude/design-references/` for specific component patterns
3. Apply these patterns when building or modifying any UI component

## When to trigger

This skill should be consulted whenever:
- A new page or component is being designed
- An existing page is being restyled or redesigned
- The user asks for UI improvements or visual changes
- Layout, spacing, color, or typography decisions are being made
